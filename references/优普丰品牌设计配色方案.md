## 优普丰品牌设计配色方案

> 4 层结构: **PRIMARY**(品牌主色) · **NEUTRALS**(中性色) · **SECONDARY**(辅助色) · **ACCENT**(点缀色)
> 严格按国际设计标准:灰色不算"颜色",是"工具集";金色管气质,橙色管行动,各司其职。

---

### 1. PRIMARY COLORS — 主色系 (蓝色系 5 色)

主色占据 60-70% 视觉量,负责品牌识别。

| HEX | 中文名 | 英文名 | 定位 |
|-----|--------|--------|------|
| `#0A2540` | 深海蓝 | Deep Sea Blue | 深色背景(封面/章节页/结尾页的整页底色) |
| `#1E4976` | 海洋蓝 | Ocean Blue | **品牌主色**(60% 视觉量) — 标题、品牌元素、关键数据 |
| `#2E6BA8` | 天空蓝 | Sky Blue | 辅助蓝 — 二级标题、卡片顶条、hover 状态 |
| `#4A8FD8` | 地平蓝 | Horizon Blue | 点缀蓝 — 图标、链接、图表高亮线条 |
| `#E8F2FC` | 冰川蓝 | Glacier Blue | 浅蓝底 — 卡片底色、表格交替行、引用块 |

### 2. NEUTRALS — 中性色 (灰色系 5 色)

灰色没有色相,只做功能性工作:文字层级、边框、背景、阴影。**不参与品牌识别,不抢蓝色位。**

| HEX | 中文名 | 英文名 | 定位 |
|-----|--------|--------|------|
| `#1A1A1A` | 碳黑 | Carbon Black | 极深文本 — 大标题(在浅底上) |
| `#3D4852` | 石板灰 | Slate Grey | 强文本 — 小标题、关键句 |
| `#6B7785` | 钢铁灰 | Steel Grey | 正文 — 主体段落(避免纯黑,更柔和专业) |
| `#A8B2BD` | 银灰 | Silver Grey | 边框/分割 — 卡片描边、表格线、分隔线 |
| `#F5F7FA` | 雾白 | Mist White | 浅色背景 — 内容页底色(与冰川蓝二选一) |

### 3. SECONDARY — 辅助色 (金色系 2 色)

金色管"气质"——高级感、奢华感、VIP 感。永远搭配深色,不要放在冰川蓝上(会糊)。同样可以用于强调和点缀的作用。

> 🔒 **铁律 #4（必须遵守）**：金色（尊贵金 `#D4AF37` / 香槟金 `#F2D492`）是金属色，**任何位置、任何尺寸、任何用途都必须用渐变**。纯色使用 = 廉价感 = 失去"高级感"（像油漆/荧光笔）。无论背景、底色、按钮、描边、文字——只要沾到金色，**100% 渐变**。下面的 hex 颜色值只用于**构造渐变的中间色**，而非"可直接当 `background-color` 用的纯色"。

| HEX | 中文名 | 英文名 | 定位（**注意：必须用渐变，不能直接用纯色**）|
|-----|--------|--------|------|
| `#D4AF37` | 尊贵金 | Prestige Gold | 金重点 — 关键数字、Logo 旁点缀、获奖标识、签名（**渐变背景**）|
| `#F2D492` | 香槟金 | Champagne Gold | 金柔光 — 浅底上的金色文字、装饰条（**渐变背景**;深底上要改用尊贵金）|

#### 🎨 金色渐变描述

**尊贵金 Prestige Gold 渐变**
- 基础色值: `#D4AF37`
- 线性渐变(Linear Gradient),方向 `135°`(左上到右下)
- 色标:
  - `0%` → `#E8C547` (亮金高光)
  - `50%` → `#D4AF37` (标准金)
  - `100%` → `#B8941F` (深金阴影)
- 金属光泽效果:
  - 在 50% 位置叠加白色高光 `rgba(255,255,255,0.3)`
  - 边缘添加深色描边 `#9A7B1A` (1px)

**香槟金 Champagne Gold 渐变**
- 基础色值: `#F2D492`
- 线性渐变,方向 `135°`
- 色标:
  - `0%` → `#FFF4D6` (柔和高光)
  - `50%` → `#F2D492` (标准香槟金)
  - `100%` → `#D9BC6E` (温暖阴影)
- 柔光效果:
  - 整体叠加柔光层 `rgba(255,248,220,0.2)`
  - 微妙光晕感,无硬边框

#### 🆕 金色文字渐变公式（`background-clip: text` 技巧）

**深色背景（海洋蓝/深海蓝）上的金色标题/关键数字**——必须用 `background-clip: text` 让字本身呈现金属光泽，绝不能写 `color: #D4AF37`（那样字会像贴上去的廉价贴纸）。

```css
/* 尊贵金文字渐变（用于深海蓝/海洋蓝背景） */
.gold-text {
  background: linear-gradient(135deg, #E8C547 0%, #F2D492 50%, #D4AF37 100%);
  -webkit-background-clip: text;
  background-clip: text;
  -webkit-text-fill-color: transparent;
  color: transparent;
}

/* 香槟金文字渐变（用于深海蓝渐变背景） */
.champagne-text {
  background: linear-gradient(135deg, #FFF4D6 0%, #F2D492 50%, #D9BC6E 100%);
  -webkit-background-clip: text;
  background-clip: text;
  -webkit-text-fill-color: transparent;
  color: transparent;
}

/* 金色描边渐变（双层 background-image 技巧） */
.gold-border {
  border: 1px solid transparent;
  background-image: linear-gradient(#1E4976, #1E4976), linear-gradient(135deg, #E8C547 0%, #B8941F 100%);
  background-origin: border-box;
  background-clip: padding-box, border-box;
}
```

#### 🆕 新增标志性奢华组合：海洋蓝背景 + 尊贵金字

深海蓝铺底 + 金属质感金色文字浮于其上，营造"深海藏金"的视觉张力——这是 UPerform 的标志性奢华组合。

| 变体 | 背景 | 文字 | 适用场景 |
|------|------|------|---------|
| 海洋蓝 + 尊贵金字 | **深海蓝渐变** `linear-gradient(135deg, #0A2540 0%, #1E4976 100%)`（必须渐变）| 尊贵金渐变文字（`#E8C547` → `#F2D492` → `#D4AF37`，`background-clip: text`）| 高端封面、Hero 标题、奖项大数字、VIP 标识、限量版标签 |
| 深海蓝 + 香槟金字 | **深海蓝渐变** `linear-gradient(135deg, #0A2540 0%, #1E4976 100%)` | 香槟金渐变文字（`#FFF4D6` → `#F2D492` → `#D9BC6E`）| 章节封面、引用金句、长标题 |

#### 金色使用规范 DO'S & DON'TS

**正确使用 DO:**
- 🆕 **金色必须用渐变**（`linear-gradient(135deg, #E8C547 0%, #D4AF37 50%, #B8941F 100%)` 或 香槟金渐变）
- 高端元素、VIP 标识
- 奖项证书、荣誉展示
- 重要 CTA 按钮强调（**优先替代橙色**,渐变背景）
- 品牌周年庆典素材
- 限量版产品标识
- 🆕 **金字用 `background-clip: text` 渐变**（深色背景上的标题、关键数字）
- 🆕 **金描边用双层 `background-image` 渐变**（小元素金属边框）
- 🆕 **强调与点缀**——关键标签、活跃高亮、数据亮点（承担原橙色职责）
- 🆕 **海洋蓝背景 + 尊贵金字**（标志性奢华组合，用于高端封面、Hero、奖项、VIP 标识）
```css
  /* 🆕 海洋蓝 + 尊贵金字卡片 (标志性奢华组合) */
  .card-ocean-gold {
    background: linear-gradient(135deg, #0A2540 0%, #1E4976 100%);
    border-left: 4px solid transparent;
    background-image: linear-gradient(135deg, #0A2540 0%, #1E4976 100%), var(--up-gold-gradient-border);
    background-origin: border-box;
    background-clip: padding-box, border-box;
    border-radius: 0 8px 8px 0;
    padding: 16px 20px;
    box-shadow: 0 4px 16px rgba(10, 37, 64, 0.3);
  }
```

**避免使用 DON'T:**
- 🆕 **使用纯色金色**（`background: #D4AF37` / `background: #F2D492` / `color: #D4AF37`）——**任何位置、任何尺寸、任何用途一律禁止**（最严重反模式）
- 🆕 **把金色文字当纯色用**（如 `color: #D4AF37`）——必须 `background-clip: text` 渐变,否则字会"贴不住"深色背景
- 大面积背景色（即使是渐变,金色作为主背景超过 30% 视觉量也会显廉价）
- 正文字颜色（金色文字只用于标题、关键数字、Hero 字,不用于正文段落）
- 低对比度组合
- 过度使用(全场不超过 3 处)
- 与竞品金色混淆
- 不在冰川蓝上使用金色（会糊）
- 不在浅色背景上使用金色文字（对比度不足,看不见）

### 4. ACCENT — 点缀色 (橙色系 2 色)

橙色是"放大镜",不是"油漆"——**全场不超过 1 处橙色元素**(按钮 + 1-2 个关键数据),多了就失去强调意义。

| HEX | 中文名 | 英文名 | 定位 |
|-----|--------|--------|------|
| `#FF6B35` | 活力橙 | Energy Orange | 活力强调 — CTA 按钮、增长率↑、关键行动点 |
| `#FF9B7D` | 珊瑚光 | Coral Glow | 柔强调 — 次级 CTA、标签 chip、暖色图标底 |

> 🆕 **金色一律使用渐变** —— 下表所有"金色"位置必须用渐变；标题用了金色作为文字色时，必须用 `background-clip: text` 渐变技术，绝不能 `color: #D4AF37` 写死。

## 四条使用铁律 (CRITICAL RULES)

1. **橙色是"放大镜",不是"油漆"**
   整场不超过 1 处橙色元素(按钮 + 1-2 个关键数据),多了就失去强调意义。金色系同样可以用于点缀和强调,橙色系尽可能地少用。

2. **金色永远搭配深色**
   尊贵金 + 深海蓝 = 经典奢华;香槟金 + 雾白 = 轻柔优雅。
   **金色不要放在冰川蓝上(会糊)**。
   金色已承担原橙色的强调与点缀职责,使用上限 ≤ 3 处。

3. **文字层级用灰色系统,蓝/橙/金不参与正文**
   正文只走 `#1A1A1A` → `#3D4852` → `#6B7785` 的灰阶。
   蓝是装饰色,不是文字色——避免大段海洋蓝文字,既疲劳又掉档次。

4. **🆕 金色一律使用渐变,禁止纯色**
   尊贵金 `#D4AF37` 与香槟金 `#F2D492` 是金属色,纯色使用 = 廉价感 = 像油漆/荧光笔。
   **所有金色元素(背景/底色/按钮/描边/文字)100% 渐变**。
   - 渐变背景:`linear-gradient(135deg, ...)` 多色渐变
   - 渐变文字:`background-clip: text` + `-webkit-text-fill-color: transparent`
   - 渐变描边:双层 `background-image` + `background-origin: border-box`
   - 大面积背景(卡片底色、Banner、Header、章节页)尤其禁止使用纯金色——这是高级感的灵魂。

## 4 层体系价值说明

很多品牌的色板只有"主色 + 辅色 + 点缀"3 层,文字层级只能硬塞进主色或辅色里,导致主色被滥用、文字蓝得发亮、辅色又抢戏。

**优普丰这套色板把 Neutrals 单独成体系**,文字/边框/背景各有 5 阶灰可用:
- **主色**就专管品牌(60-70% 视觉量)
- **灰色**就专管功能(文字/边/底)
- **金色**管气质(高级感)
- **橙色**管行动(CTA 按钮)

各司其职,主色不被稀释、文字不疲劳、CTA 不抢戏。

## 对外介绍建议格式

```
PRIMARY        NEUTRALS                 SECONDARY (渐变)     ACCENT
海洋蓝         碳黑/石板/钢铁/银/雾白  尊贵金渐变/香槟金渐变  活力橙/珊瑚光
```

设计师 / 前端 / 市场一看就懂:
- 灰色是工具 ✏️
- 金色是气质 👑 (**必须用渐变**,禁用纯色)
- 橙色是按钮 🚀
- 蓝色是品牌 🎨

> 🆕 **备忘**:金色 = 渐变;金字 = `background-clip: text` 渐变;金底 = 深海蓝渐变 + 金字渐变(标志性奢华)。
